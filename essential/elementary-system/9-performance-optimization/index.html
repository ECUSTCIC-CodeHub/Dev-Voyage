第九章：电脑卡了？先别砸

约 4162 字大约 14 分钟

2025-12-22

历史趣事：安迪-比尔定律

在计算机界有一个著名的定律叫“安迪-比尔定律”（Andy and Bill's Law）。

安迪是指英特尔的前CEO安迪·格鲁夫，比尔是指微软的比尔·盖茨。定律的内容是：“安迪提供什么，比尔就拿走什么。”（What Andy gives, Bill takes away.）

意思是：英特尔辛辛苦苦把硬件性能提升了一倍，微软（以及其他软件厂商）就会开发出更庞大、更吃资源的软件，把这些新增的性能全部吃掉。

这就是为什么你的电脑明明比十年前快了100倍，但打开Word的速度感觉还是差不多。

问题来了：“为什么新电脑用半年就变卡？那些‘电脑管家’到底有没有用？什么时候该重装系统？”

一、一个真实的“电脑卡顿”惨案

真实案例：8000块的电脑用出800块的感觉

上周，小王的游戏本：

  1. 刚买时：开机10秒，游戏120帧，流畅如丝 ✅
  2. 三个月后：开机1分钟，游戏掉到60帧，有点卡 😕
  3. 半年后：开机3分钟，游戏30帧，卡成PPT 😫
  4. 现在：打开Word都要等10秒，风扇狂转像直升机 🚁

他做了什么？

  • 装了3个“电脑管家”
  • 桌面放了500多个文件
  • C盘只剩2GB空间
  • 开机自启动30多个程序

结果：花了8000块买的电脑，用出了800块的感觉。

二、电脑为什么会变卡？

2.1 硬件老化？不完全是

一个常见误解： “电脑用久了，硬件会老化变慢” ❌

真相是：

  • CPU、内存、硬盘这些硬件，性能不会随时间下降
  • 但软件越来越吃资源，硬件相对就“变弱”了
  • 就像10年前的车还能开，但上不了今天的高速公路

电脑卡顿的四大元凶

第一名：存储空间不足（40%的卡顿原因）

  • 症状：开机慢，打开软件慢，保存文件慢
  • 原理：硬盘需要留出15-20%的空白空间作为“缓冲区”
  • 临界点：C盘剩余空间 < 总容量10%

第二名：后台程序太多（30%原因）

  • 症状：什么都没干，风扇狂转，内存占用80%+
  • 元凶：各种软件的“开机自启动”、“后台服务”、“自动更新”
  • 典型代表：Adobe Creative Cloud、Google Update、腾讯QQProtect

第三名：软件“膨胀”（20%原因）

  • 症状：同样的软件，新版比旧版占用更多资源
  • 例子：
    • Chrome浏览器：2010年内存占用200MB，现在开5个标签就1GB
    • 微信：从单纯的聊天工具变成“操作系统”
    • Office：功能越来越多，启动越来越慢

第四名：散热问题（10%原因）

  • 症状：用一会儿就卡，特别是夏天或玩游戏时
  • 原理：CPU过热会自动降频保护（从3.0GHz降到1.5GHz）
  • 常见场景：笔记本放在床上用堵了进风口，几年没清灰
  • 原理：CPU过热会自动降频保护（从3.0GHz降到1.5GHz）
  • 常见场景：笔记本放在床上用堵了进风口，几年没清灰

三、存储空间：C盘红了怎么办？

3.1 各系统查看存储空间

Windows：

  1. 按Win + E打开文件资源管理器
  2. 在“此电脑”里看C盘剩余空间
  3. 详细分析：设置 → 系统 → 存储 → 点击C盘

macOS：

  1. 点击左上角苹果标志 → 关于本机 → 存储
  2. 详细分析：关于本机 → 存储 → 管理

Linux：

df -h  # 查看所有磁盘空间
du -sh *  # 查看当前文件夹大小

手机（安卓/iOS）：

  • 设置 → 通用 → iPhone存储空间 / 设置 → 存储

3.2 快速清理：这些可以删

Windows用户必删清单：

  1. 临时文件：C:\Windows\Temp 和 %TEMP%（按Win+R输入%TEMP%）
  2. 下载文件夹：C:\Users\你的名字\Downloads
  3. 旧系统备份：设置 → 系统 → 存储 → 临时文件 → 以前的Windows安装
  4. 休眠文件（如果你不用休眠）：以管理员运行CMD：powercfg -h off
  5. 回收站：右键回收站 → 清空回收站

macOS用户必删清单：

  1. 下载文件夹：~/Downloads
  2. 废纸篓：右键Dock栏废纸篓 → 清空
  3. iOS备份：~/Library/Application Support/MobileSync/Backup
  4. Xcode缓存（如果不用Xcode）：~/Library/Developer/Xcode/DerivedData

通用清理工具：

  • Windows：自带的“磁盘清理”工具（搜索“磁盘清理”）
  • macOS：自带的“储存空间管理”
  • 第三方：BleachBit（开源免费）、CCleaner（有免费版但小心捆绑）

3.3 高级技巧：移动这些文件夹

Windows移动“文档”、“下载”、“桌面”：

  1. 右键“文档”文件夹 → 属性 → 位置
  2. 点击“移动”，选择D盘或其他分区的新位置
  3. 系统会问是否移动现有文件，选“是”

原理：这些文件夹默认在C盘，把它们移到其他盘，给C盘腾空间。

macOS类似操作：

  1. 把“下载”、“文档”文件夹拖到外置硬盘
  2. 按住Cmd+Option拖，会创建替身（快捷方式）

四、后台程序：谁在偷偷吃资源？

4.1 找出“资源吸血鬼”

Windows任务管理器（按Ctrl+Shift+Esc）：

  1. 看“进程”选项卡，按“内存”或“CPU”排序
  2. 重点检查：内存占用>200MB，CPU持续>10%的程序
  3. 可疑程序：名字奇怪的.exe、各种“助手”、“更新服务”

macOS活动监视器（应用程序→实用工具）：

  1. 看“内存”和“CPU”选项卡
  2. 重点检查：%CPU持续高的进程

Linux：

top  # 实时查看进程
htop  # 更好用的top（需要安装）

4.2 禁用开机自启动

Windows 10/11：

  1. 任务管理器 → 启动选项卡
  2. 右键不需要的程序 → 禁用
  3. 保守原则：只禁用你认识的非必要程序

macOS：

  1. 系统设置 → 通用 → 登录项
  2. 删除不需要开机启动的程序

Windows更彻底的方法（msconfig）：

  1. 按Win + R，输入msconfig
  2. 服务选项卡 → 勾选“隐藏所有Microsoft服务”
  3. 剩下的可以酌情禁用（但要知道在做什么）

Linux（Ubuntu）：

sudo systemctl list-unit-files --type=service  # 查看所有服务
sudo systemctl disable 服务名  # 禁用服务

4.3 浏览器：内存杀手第一名

Chrome内存查看：

  1. 按Shift + Esc打开Chrome任务管理器
  2. 你会惊讶：一个标签页可能占用500MB内存

浏览器省内存技巧：

  1. 扩展程序管理：chrome://extensions/，关掉不用的
  2. 标签页管理：用OneTab等扩展把不用的标签页“冻结”
  3. 硬件加速：设置 → 系统 → 开启硬件加速（用GPU减轻CPU负担）

五、软件膨胀：如何选择轻量软件？

5.1 常见软件的“轻量替代品”

需求主流软件（重）轻量替代品（轻）
办公套件Microsoft Office（2GB+）WPS Office（200MB）、LibreOffice
图像处理Photoshop（3GB+）GIMP（300MB）、Paint.NET
视频播放暴风影音（带广告）VLC（100MB）、MPC-HC
压缩软件WinRAR（带广告）7-Zip（2MB）、Bandizip
浏览器Chrome（内存杀手）Firefox、Edge（现在其实不错）

5.2 如何判断软件是否“太重”？

安装包大小：

  • 轻量：< 50MB
  • 中等：50-200MB
  • 重量级：> 200MB

内存占用：

  • 轻量：< 100MB
  • 中等：100-500MB
  • 重量级：> 500MB

启动时间：

  • 轻量：< 3秒
  • 中等：3-10秒
  • 重量级：> 10秒

5.3 绿色软件/便携软件

概念：不需要安装，解压就能用，不写注册表，不污染系统

推荐站点：

  • PortableApps.com：各种便携软件
  • 小众软件：国内推荐绿色软件的博客

优点：

  • 不污染系统
  • 可以放在U盘里随身带
  • 卸载直接删除文件夹就行

缺点：

  • 可能没有自动更新
  • 有些功能可能缺失

六、散热问题：给电脑“退烧”

6.1 判断是否过热

Windows查看温度：

  • HWMonitor（免费软件）
  • Core Temp（专门看CPU温度）

macOS：

  • iStat Menus（付费但强大）
  • Fanny（免费，在通知中心显示温度）

临界温度：

  • CPU：> 85°C 开始降频，> 95°C 危险
  • GPU：> 80°C 开始降频，> 90°C 危险

6.2 物理清灰（胆小勿试）

笔记本清灰步骤：

  1. 准备工具：螺丝刀、刷子、吹风机（冷风档）
  2. 断电：拔电源，拆电池（如果可拆）
  3. 拆后盖：找到散热风扇位置
  4. 清灰：用刷子轻轻刷，吹风机吹走灰尘
  5. 装回去：螺丝别弄丢，别拧太紧

如果不敢拆：

  1. 用吹风机冷风档吹散热口
  2. 买个散热底座（有点用，但有限）
  3. 找专业人士清灰（50-100元）

6.3 软件“降温”

降低性能换温度：

  • Windows电源计划：设置 → 系统 → 电源和睡眠 → 其他电源设置 → 选择“节能”
  • 游戏本控制中心：大多数游戏本有控制软件，可以调性能模式
  • 限帧：游戏里限制最高帧率（比如从144帧限到60帧）

七、重装系统：电脑的“重启人生”

7.1 什么时候该重装？

重装系统的正确时机：

  1. 系统崩溃，无法正常启动
  2. 中了严重病毒，杀不干净
  3. 用了3年以上，积累了太多垃圾
  4. 卖电脑前清空数据

不该重装的时候：

  1. 只是有点卡（先试试清理）
  2. 某个软件问题（重装那个软件就行）
  3. 没有备份重要数据（先备份！）

7.2 各系统重装方法

Windows 10/11重置（最安全）：

  1. 设置 → 更新和安全 → 恢复
  2. 选择“重置此电脑”
  3. 关键选择：
    • 保留我的文件：只重装系统，保留个人文件
    • 删除所有内容：彻底清空（卖电脑前用这个）

macOS恢复：

  1. 关机，按Cmd+R开机进入恢复模式
  2. 选择“重新安装macOS”
  3. 注意：需要苹果ID和网络连接

Linux重装：

# 其实Linux很少需要重装，但如果你坚持：
# 制作启动U盘，从U盘启动安装

7.3 重装前的准备清单

必须备份的：

  1. 桌面、文档、下载文件夹
  2. 浏览器书签（登录账号同步最简单）
  3. 重要软件的配置/存档：
    • 微信聊天记录（用电脑版备份）
    • 游戏存档（Steam云存档会自动同步）
    • 开发环境配置（如果有）

备份工具推荐：

  • 文件：直接用U盘/移动硬盘拷贝
  • 系统镜像：Macrium Reflect（Windows）、Time Machine（macOS）

八、历史趣事：从“碎片整理”到“SSD革命”

8.1 机械硬盘时代的“碎片整理”

什么是磁盘碎片？

  • 早期机械硬盘：文件被分成小块，散落在磁盘各处
  • 读取时：磁头需要到处移动，速度变慢
  • 整理：把文件碎片拼在一起，让磁头少跑路

Windows的经典画面：

  • Windows 98/XP时代：碎片整理进度条能看一小时
  • 人们真的会看着进度条发呆
  • 整理一次能提升20-30%速度

SSD时代的变革：

  • SSD没有磁头，不存在“碎片”问题
  • 碎片整理对SSD有害（减少寿命）
  • Windows 10/11：对SSD自动禁用碎片整理

8.2 内存价格的“过山车”

历史价格：

  • 1990年：1MB内存 ≈ 500美元（今天价值1000美元）
  • 2000年：128MB内存 ≈ 100美元
  • 2010年：4GB内存 ≈ 100美元
  • 2020年：16GB内存 ≈ 100美元（疫情期间涨过）

有趣现象：

  • 比尔·盖茨1981年说“640KB内存对任何人都够了”
  • 现在：Chrome开10个标签就用掉4GB
  • 软件开发者知道内存便宜了，就写“浪费内存”的代码

8.3 摩尔定律的“减速”

摩尔定律（1965年提出）： 每18-24个月，芯片上的晶体管数量翻一倍

曾经的效果：

  • 1995-2005：CPU速度每年提升50%
  • 2005-2015：提升降到20%
  • 2015-现在：提升降到10%

为什么变慢？

  • 物理极限：晶体管不能再小了（现在3纳米）
  • 散热问题：频率不能再高了（现在5GHz是极限）
  • 电力限制：手机电池撑不住更快的芯片

结果：现在电脑性能提升主要靠：

  1. 更多核心（从单核到8核、16核）
  2. 专用芯片（GPU、NPU、TPU）
  3. 软件优化

九、防卡顿的日常习惯

9.1 安装软件时的“三不原则”

不安装：不认识、不需要、不信任的软件 不勾选：安装时的附加选项（浏览器主页、捆绑软件） 不放C盘：尽量安装到D盘或其他分区

9.2 每月一次的“电脑体检日”

每月第一个周末：

  1. 清理临时文件（各系统方法前面讲了）
  2. 更新软件（但不是立刻更新系统大版本）
  3. 检查启动项（有没有新增的）
  4. 备份重要文件（到移动硬盘或云盘）

9.3 硬件升级：最有效的提速方法

升级优先级：

  1. 加内存（最有效）：4GB→16GB，感受明显
  2. 换SSD（第二有效）：机械硬盘→固态硬盘，开机从1分钟变10秒
  3. 换CPU/显卡（最贵）：通常不如整机换新

如何判断该升级什么？

  • 任务管理器里：内存经常>80% → 加内存
  • 打开文件慢，开机慢 → 换SSD
  • 玩游戏卡 → 加内存或换显卡

十、本章实操任务

🎯 必做任务（今天！）：

  1. 检查你的C盘/系统盘：
    • Windows：设置 → 系统 → 存储，看什么占用了最多空间
    • macOS：关于本机 → 存储 → 管理
    • 删除至少2GB的垃圾文件
  2. 清理开机启动项：
    • 禁用至少3个不需要开机启动的程序
    • 重启电脑，感受开机速度变化
  3. 检查温度（可选）：
    • 下载HWMonitor或类似工具
    • 看CPU/GPU温度是否正常（<80°C）

🚀 进阶挑战：

  1. 迁移个人文件夹：
    • 把“文档”、“下载”、“桌面”移到D盘（Windows）
    • 或整理这些文件夹，删除不需要的文件
  2. 创建系统还原点：
    • Windows：搜索“创建还原点”，创建一个
    • 这是你搞砸系统时的“后悔药”
  3. 制定备份计划：
    • 选择备份工具（Windows文件历史/Time Machine/第三方）
    • 设置自动备份（至少每周一次）
    • 测试恢复一个文件，确保备份有效

🤔 思考题：

“如果未来的电脑硬件性能提升缓慢，软件却越来越‘膨胀’，我们该如何平衡功能丰富性和运行效率？” （提示：云电脑、订阅制软件、模块化设计、Web应用...）

十一、避坑指南：常见问题解答

Q：电脑管家/杀毒软件有必要装吗？ A：

  • Windows 10/11自带Windows Defender，足够日常使用
  • 装一个靠谱的就行（别装多个，它们会打架）
  • 国产“全家桶”谨慎安装，可能带来更多问题
  • 推荐：火绒（国内轻量）、卡巴斯基（国际大厂）

Q：SSD需要碎片整理吗？ A：绝对不要！ SSD有写入次数限制，碎片整理会大量写入，减少寿命。Windows 10/11对SSD会自动禁用碎片整理。

Q：虚拟内存应该设多大？ A：让系统自动管理。但如果你内存很小（4GB），可以手动设置：初始大小=内存大小×1.5，最大大小=内存大小×3。

Q：清灰会把电脑搞坏吗？ A：如果小心操作，不会。但如果你没把握，花点钱找专业人士。特别是还在保修期内，自己拆可能失去保修。

Q：电脑太卡，升级还是买新的？ A：

  • 老电脑（>5年）：建议买新的，升级性价比低
  • 中等电脑（2-5年）：加内存换SSD，还能战3年
  • 新电脑（<2年）：检查是不是软件问题，优化一下

Q：重装系统会丢失所有软件吗？ A：

  • “重置此电脑”选择“保留我的文件”：保留个人文件，但软件需要重装
