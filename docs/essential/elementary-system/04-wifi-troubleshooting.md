## 📡 Wi-Fi 到底是怎么工作的？

### 简单说：Wi-Fi 就是"无线电波传数据"

**类比：**
- 路由器 = 一个不停喊话的大喇叭
- 你的设备 = 带着耳机听喇叭的人
- Wi-Fi 信号 = 空气中传播的声波

**实际过程：**
1. 路由器广播："我叫'TP-Link_隔壁老王'，频率 2.4GHz，密码是'12345678'"
2. 你的手机说："我收到信号了，要连接'隔壁老王'，密码是'12345678'"
3. 路由器说："密码正确，给你分配一个座位号（IP 地址），比如'192.168.1.101'"
4. 连接成功！现在你可以上网了

### 神秘的 2.4GHz 和 5GHz

现在的路由器通常有两个频段，就像两条不同的车道：

::: info 2.4G vs 5G：选哪个？
| 特性 | 2.4GHz 频段 | 5GHz 频段 |
|:---|:---|:---|
| **速度** | 慢（乡间小路） | 快（高速公路） |
| **穿墙能力** | 强（能绕过障碍物） | 弱（遇到墙就衰减） |
| **干扰** | 多（蓝牙、微波炉都用这个） | 少（专属车道） |
| **适用场景** | 离路由器远、隔墙多 | 离路由器近、打游戏、看 4K 视频 |

**建议**：能连 5G 就连 5G，连不上再连 2.4G。
:::
- **缺点**：穿墙能力弱，覆盖范围小
- **适合**：距离近、需要高速的场景（打游戏、看 4K 视频）

**你的设备支持哪个？**
- 2015 年后的手机/电脑基本都支持 5GHz
- 看路由器：如果有两个 Wi-Fi 名称（如 `TP-Link_5G`），就是双频路由器

::: tip 实用技巧：如何选择频段？
- **打游戏、看高清视频** → 连 5GHz（速度优先）
- **上厕所刷手机** → 连 2.4GHz（覆盖优先）
- **智能家居设备**（扫地机器人、智能灯）→ 通常只支持 2.4GHz
:::

## 🔧 为什么手机能连，电脑不行？（灵魂拷问）

### 常见原因排名

**第一名：IP 地址冲突（40% 概率）**
- **症状**：显示"已连接，无 Internet"
- **原因**：两台设备被分到了同一个 IP 地址（就像电影院两个观众拿了同一张票）
- **解决**：重启路由器或电脑，让路由器重新分配 IP

**第二名：网卡驱动过时（30% 概率）**
- **症状**：完全搜不到 Wi-Fi，或者信号满格但速度极慢
- **原因**：电脑的 Wi-Fi 模块"翻译官"太老了，听不懂路由器的新方言
- **解决**：更新网卡驱动（后面教你怎么做）

**第三名：电源管理捣鬼（20% 概率）**
- **症状**：用着用着突然断线，过会儿又自动连上
- **原因**：Windows 为了省电，让网卡间歇性"睡觉"
- **解决**：关闭网卡的电源管理（后面有教程）

**第四名：路由器限制（10% 概率）**
- **症状**：特定设备永远连不上
- **原因**：路由器设置了 MAC 地址过滤（只让认识的设备连）
- **解决**：登录路由器后台，把这个设备加入白名单

### 诊断流程图：跟着做就能解决

```
电脑连不上 Wi-Fi？别慌！
↓
第一步：重启路由器（拔电源等 30 秒）
↓
还不行？ → 第二步：重启电脑
↓
还不行？ → 第三步：忘记网络重新连接
↓
还不行？ → 第四步：更新网卡驱动
↓
还不行？ → 第五步：检查 IP 设置（设为自动获取）
↓
还不行？ → 可能是硬件问题，考虑外接 USB 网卡
```

## 🖥️ 各系统网络设置指南

### Windows 用户专属

**查看网络状态：**
1. 右下角网络图标 → 右键 → "打开网络和 Internet 设置"
2. 点击"更改适配器选项"
3. 右键"WLAN" → 状态 → 看"速度"（72Mbps 以上算正常）

**更新网卡驱动：**
1. 右键"开始菜单" → 设备管理器
2. 展开"网络适配器"
3. 右键你的无线网卡（通常带"Wireless"、"Wi-Fi"字样）
4. 选择"更新驱动程序" → "自动搜索"

**关闭省电模式：**
1. 在设备管理器里，右键无线网卡 → 属性
2. 切换到"电源管理"选项卡
3. **取消勾选**"允许计算机关闭此设备以节约电源"

**网络重置（大招）：**
设置 → 网络和 Internet → 状态 → 网络重置
⚠️ **警告**：这会删除所有 Wi-Fi 密码和网络设置，慎用！

### macOS 用户专属

**查看网络详情：**
1. 按住 Option 键，点击菜单栏的 Wi-Fi 图标
2. 能看到信号强度（RSSI）、信道、传输速率等详细信息

**删除网络配置文件：**
1. 系统设置 → Wi-Fi
2. 点击"高级"
3. 找到有问题的网络 → 点击"-"号删除
4. 重新连接

**终端命令修复：**
```bash
# 重新获取 IP 地址
sudo ipconfig set en0 DHCP

# 刷新 DNS 缓存
sudo dscacheutil -flushcache
```

### 安卓/iOS 用户专属

**安卓 Wi-Fi 修复：**
1. 设置 → 网络和 Internet → Wi-Fi
2. 长按连接的 Wi-Fi → 修改网络
3. 点击"IP 设置" → 从"DHCP"改为"静态"，再改回"DHCP"

**iOS Wi-Fi 修复：**
1. 设置 → Wi-Fi → 点击当前 Wi-Fi 后面的"i"图标
2. 点击"忘记此网络"
3. 重新连接

**手机当临时热点：**
当你电脑连不上 Wi-Fi 但手机可以时：
1. 手机开启"个人热点"
2. 电脑连接手机热点上网
3. 趁能上网时，下载驱动或查找解决方案

## 🎛️ 路由器：你家网络的"总司令"

### 怎么进路由器后台？

**第一步：找到路由器 IP**
- Windows：按 `Win + R`，输入 `cmd`，回车，输入 `ipconfig`，看"默认网关"
- macOS：系统设置 → 网络 → 高级 → TCP/IP，看"路由器"
- 通常是：`192.168.1.1`、`192.168.0.1`、`192.168.50.1`

**第二步：登录后台**
1. 浏览器输入路由器 IP
2. 输入用户名密码（通常在路由器底部贴纸上）
3. 常见组合：admin/admin、admin/123456、admin/password

**第三步：关键设置检查**
1. **Wi-Fi 名称和密码**：可以在这里改
2. **信道**：2.4GHz 选 1、6、11；5GHz 选 149 以上（干扰少）
3. **频宽**：2.4GHz 选 20MHz；5GHz 选 80MHz（速度更快）
4. **DHCP 服务器**：确保开启，IP 池范围够大

::: warning ⚠️ 不要乱改这些！
- WAN 口设置（改了可能上不了网）
- 运营商信息（光猫拨号信息）
- 恢复出厂设置（除非你知道自己在做什么）
:::

### 路由器摆放的玄学

**黄金位置：**
- 房子中央，高度 1.5 米左右（桌面或柜子）
- 远离金属物体（冰箱、微波炉）
- 远离承重墙和镜子
- 天线竖直向上（信号水平扩散）

**错误示范：**
- 塞在弱电箱里 ❌（金属门屏蔽信号）
- 放在地上 ❌（信号被家具遮挡）
- 藏在电视后面 ❌（电视也产生干扰）

**增强信号小技巧：**
1. 用易拉罐 DIY 信号放大器（网上有教程，效果有限但好玩）
2. 购买 Wi-Fi 信号放大器（中继器）
3. 组 Mesh 网络（多个路由器协同工作，土豪之选）

## 📜 历史趣事：从"无线电报"到 Wi-Fi 7

### Wi-Fi 名字的由来

**一个美丽的误会：**

很多人都以为 Wi-Fi 是"Wireless Fidelity"（无线保真）的缩写，就像 Hi-Fi（高保真）一样。

**实际真相：**

Wi-Fi 只是一个品牌名称，由品牌咨询公司 Interbrand 创造，本身没有具体含义。当时 Wi-Fi 联盟想要一个比"IEEE 802.11b Direct Sequence"更亲民的名字。

**命名趣事：**

据说 Interbrand 提供了 10 个选项，Wi-Fi 是其中一个。Wi-Fi 联盟的成员觉得它听起来有点像 Hi-Fi，容易让人联想到高质量，就选了它。

### 历代 Wi-Fi 标准

| 标准 | 年代 | 最大速度 | 绰号 | 特点 |
|:---|:---|:---|:---|:---|
| 802.11 | 1997 | 2 Mbps | 老祖宗 | 比拨号上网还慢 |
| 802.11b | 1999 | 11 Mbps | 第一代普及 | 开始进入家庭 |
| 802.11g | 2003 | 54 Mbps | 经典一代 | 很多老设备还在用 |
| 802.11n | 2009 | 600 Mbps | Wi-Fi 4 | 引入 MIMO（多天线） |
| 802.11ac | 2013 | 1.3 Gbps | Wi-Fi 5 | 5GHz 频段普及 |
| 802.11ax | 2019 | 9.6 Gbps | Wi-Fi 6 | 更省电，多设备优化 |
| 802.11be | 2024+ | 40 Gbps | Wi-Fi 7 | 还没完全普及 |

::: tip 有趣的事实
- Wi-Fi 6 的速度比最初的 802.11 快了 **4800 倍**
- 但很多人家里的实际网速只有 100Mbps（因为宽带限制）
- 就像买了跑车却在市区开，根本跑不起来

**所以**：路由器再快，也要看看你的宽带套餐是多少！
:::

## 📱 移动网络：4G/5G 的真相

### 为什么手机信号有时候满格却没网？

**三大原因：**
1. **基站拥堵**：太多人同时用（比如演唱会现场）
2. **信号虚标**：手机显示的是信号强度，不是网络质量
3. **限速**：运营商给"不限量套餐"用户限速（超过一定流量后变慢）

**改善方法：**
- 开启飞行模式 10 秒再关闭（强制重新连接基站）
- 切换到 3G 网络试试（有时候 4G 挂了 3G 还能用）
- 投诉运营商（真的有用）

### 5G 真的那么神吗？

**5G 的优势：**
- 低延迟：打游戏几乎感觉不到延迟
- 高速度：理论速度比 4G 快 10-100 倍
- 大连接：能同时连接更多设备（物联网基础）

**5G 的缺点：**
- 耗电快：手机续航明显下降
- 覆盖差：基站密度需要是 4G 的 3-5 倍
- 穿透弱：进建筑物信号衰减严重

**残酷的现实：**

很多人用着 5G 手机，开着 5G 套餐，实际速度跟 4G 差不多，因为：
1. 基站还没完全建好
2. 运营商偷偷限速
3. 手机发热自动降频

## 🔒 网络安全：公共 Wi-Fi 能不能用？

### 公共 Wi-Fi 的风险

**三大危险：**
1. **中间人攻击**：黑客伪装成 Wi-Fi，窃取你的数据
2. **嗅探攻击**：黑客监听同一 Wi-Fi 下的所有流量
3. **恶意热点**：Wi-Fi 名称可能是"Starbucks_Free"，其实是黑客设的陷阱

**哪些操作绝对不能在公共 Wi-Fi 做：**
- 登录网银、支付宝、微信支付
- 输入信用卡信息
- 登录重要账号（邮箱、社交账号）

### 安全使用指南

**如果必须用公共 Wi-Fi：**
1. 向工作人员确认正确的 Wi-Fi 名称
2. 使用 VPN（虚拟专用网络）加密所有流量
3. 开启手机的"随机 MAC 地址"功能（防止追踪）
4. 用完立即"忘记此网络"

::: warning 免费 VPN 慎用！
很多免费 VPN：
- 偷偷卖你的数据
- 植入广告甚至恶意软件
- 速度慢到怀疑人生

**付费 VPN 选知名品牌**：ExpressVPN、NordVPN、Surfshark
:::

## ❓ 你可能遇到的问题

**Q：晚上 8-11 点网速特别慢怎么办？**

A：这是"晚高峰"，整栋楼都在上网。解决方案：
- 联系运营商升级带宽
- 使用 5GHz 频段（用的人少）
- 错峰下载（设置凌晨自动下载）

---

**Q：显示已连接但打不开网页？**

A：DNS 问题。解决方法：
1. 电脑：网络设置 → 更改适配器选项 → 右键 Wi-Fi → 属性
2. 双击"IPv4" → 使用以下 DNS：`8.8.8.8` 和 `114.114.114.114`

---

**Q：路由器要多久重启一次？**

A：
- 普通家用：每周重启一次
- 多人合租：每 3 天重启一次
- 打游戏专用：每次玩之前重启

---

**Q：Wi-Fi 辐射有害吗？**

A：Wi-Fi 辐射功率约 0.1 瓦，手机通话时辐射约 1-2 瓦。相比之下，阳光的辐射功率约 1000 瓦/平方米。所以，**Wi-Fi 辐射还没阳光辐射的万分之一强**，不用担心。

---

**Q：Mesh、中继器、电力猫选哪个？**

A：
- **单间信号差** → 中继器（最便宜）
- **多楼层/大户型** → Mesh（最稳定但贵）
- **无法布线且 Wi-Fi 穿墙差** → 电力猫（通过电线传输网络）

---

**Q：怎么查看连接了 Wi-Fi 的设备？**

A：登录路由器后台，找到"设备列表"或"已连接设备"，可以看到所有连接到 Wi-Fi 的设备。如果发现陌生设备，立即修改 Wi-Fi 密码！

---

**Q：Wi-Fi 密码忘了怎么办？**

A：
1. 电脑已连接过：查看 → 网络连接 → Wi-Fi 属性 → 显示字符
2. 路由器底部贴纸有密码
3. 登录路由器后台查看
4. 如果都不行，只能重置路由器（会断网，需要重新设置）
