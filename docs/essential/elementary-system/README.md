# 🚀 别慌！计算机其实很好懂

> 给数字时代新手的生存指南 —— 让你的手机和电脑不再神秘兮兮！

::: tip 欢迎来到数字世界！
如果你符合以下任何一条：
- 手机玩游戏贼6，打开电脑却连文件都找不到
- 每次装软件都战战兢兢，怕点到什么奇怪的东西
- 不明白为什么同样一个 Wi-Fi，手机能连电脑不行
- 对"系统更新"有种莫名的恐惧
- 觉得会用命令行的人都是黑客

那么，这门课就是为你写的！🎉
:::

## 🎯 这门课适合谁？

::: note 目标人群
- 会用手机，但电脑操作一脸懵的你
- 工作中不得不和电脑打交道，但又不好意思问同事
- 想学编程但连"文件夹"都搞不清楚
- 对技术有恐惧症，觉得自己"学不会这个"
- 好奇宝宝想知道"为什么电脑要这样设计"
:::

**好消息**：你不需要任何基础！这门课假设你唯一的数字设备经验是手机——而且只是会用，不明白原理。

## ✨ 你将获得什么？

::: tip 课程收获
1. **不再恐惧技术** — 电脑卡了知道怎么办，Wi-Fi 连不上知道怎么修
2. **效率翻倍** — 用电脑完成工作，速度至少快一倍
3. **省钱** — 会自己解决小问题，不用每次都找"懂电脑的朋友"
4. **安全意识** — 知道如何保护自己的数据和隐私
5. **坚实基础** — 如果你想学编程、做设计、剪视频，这些知识都是地基

## 🎮 课程特色

每章都从一个**真实场景**开始，让你看到自己的影子：
- "文件明明保存了，怎么找不到了？"

- "我只是点了一下安装，电脑怎么多了三个浏览器？"

### 📚 覆盖所有主流系统

::: note 系统覆盖
- **Windows** — 办公室最常见，你的同事都在用
- **macOS** — 苹果用户的优雅选择
- **Linux** — 技术大佬的最爱，其实没那么可怕
- **安卓/iOS** — 你的手机，我们用它来类比理解电脑

换设备也不慌，核心概念都是相通的！
:::

## 💬 常见问题

::: note Q&A
**Q：我真的零基础，能学会吗？**

A：这门课就是为纯小白设计的。我们假设你唯一的数字设备经验是手机。跟着教程走，不懂就多看几遍，你会发现计算机其实很讲道理。

**Q：需要买什么特别的设备吗？**

A：完全不用！有手机 + 任意一台电脑就够了。没有电脑？用手机也能学懂大部分概念。

**Q：学完能成为程序员吗？**

A：不能直接成为程序员，但如果你想学编程，这是最好的起点。我们会给你打下扎实的基础，让你后面学编程时不会一脸懵。

**Q：为什么不用更"专业"的术语？**

A：因为专业术语是学习的结果，不是起点。我们先帮你理解概念，你自然就能学会术语。而且，专业术语记不住没关系，用人话描述清楚意思才是关键！

**Q：遇到问题怎么办？**

A：每章都有"你可能遇到的问题"板块。如果还有困惑，可以搜索（这是程序员的基本技能！），或者在项目里提 Issue。
:::
